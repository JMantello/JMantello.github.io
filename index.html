Projects

Secret Recipe API - GitHub Repository

Social platform for food enthusiasts - RESTful API with Kotlin, Spring Boot, JWT, Grafana/Prometheus.

Description

A RESTful social platform for food enthusiasts to share and discover recipes. Built with Kotlin and Spring Boot, it allows users to publish, edit, and delete recipes, and interact with other users by following them, saving recipes, and leaving reviews. It's designed for scalability and flexibility while supporting a broad range of social interactions among users.

Technologies

The project is built with Kotlin and Spring Boot for endpoints and security, JWT for authentication and authorization, Grafana and Prometheus for monitoring and metrics, Docker Compose for orchestrating the containerized services, K6 for load-testing, and Postman for live-documentation and testing the API.


Spaceflight News UI - GitHub Repository

For browsing news articles, blogs, and reports related to spaceflight and launch events.

Description

A React web application for browsing news articles, blogs, and reports related to spaceflight and launch events. It's integrated with the Spaceflight News API, a free-to-use, public API that provides up-to-date spaceflight news. Designed to be responsive and accessible, and it provides a clean and intuitive user interface for browsing and reading spaceflight news.

Technologies

The project is built with React, JavaScript, and Tailwind CSS to desgin an intuitive and responsive user interface, including accessible themes and components, search and filter functionality, and a home page with the latest news articles.
