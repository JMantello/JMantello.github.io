Projects

Secret Recipe API - GitHub Repository

Social platform for food enthusiasts - RESTful API with Kotlin, Spring Boot, JWT, Grafana/Prometheus.

Description

A RESTful social platform for food enthusiasts, enabling the sharing, discovering, and discourse of recipes. Built with Kotlin and Spring Boot, it allows users to manage recipes and interact with users through following, liking, saving, and reviewing recipes. The project was an exercise in creating a scalable and flexible API that can support a broad range of functions using best practices and design patterns, and building production readiness with features like authentication, documentation, testing, and monitoring.

Technologies

The project is built with Kotlin and Spring Boot foundation API, JWT authentication and authorization, Grafana and Prometheus monitoring and metrics, Docker Composer container orchestration of services, K6 load-testing, and Postman live-documentation and testing.


Spaceflight News UI - GitHub Repository

For browsing news articles, blogs, and reports related to spaceflight and launch events.

Description

A React web application for browsing news articles, blogs, and reports related to spaceflight and launch events. It's integrated with the Spaceflight News API which is a free-to-use, public API that provides up-to-date spaceflight news. Designed to be responsive and accessible, and it provides a clean and intuitive user interface for browsing and reading spaceflight news.

Technologies

The project is built with React, JavaScript, and Tailwind CSS to desgin an intuitive and responsive user interface, including accessible themes and components, search functionality, and a home page with the latest news articles.
